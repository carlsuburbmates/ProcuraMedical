## Day-1 Operations (What to Expect)

**First order arrives (1â€“5 orders expected in first week):**

1. **Customer places order** (website or email)
2. **You create Proforma Invoice** (doc 22 template):
   - Fill in customer details, product, price, NDIS code (if plan-managed)
   - If plan-managed: email proforma to plan manager for approval
3. **Payment received** (or approval from plan manager):
   - Create Final Invoice (doc 23)
   - Email to customer + plan manager
4. **Log order immediately** in Order_Log (doc 15):
   - Order ID, date, customer, supplier, SKU, status "Pending"
5. **Contact supplier** (email or system):
   - Request blind shipment to customer address
   - Provide tracking when supplier sends
6. **Update Order_Log** when:
   - [ ] Supplier confirms dispatch
   - [ ] Customer receives (get confirmation)
   - Add tracking #, delivery date

**If anything goes wrong (damage, delay, defect):**
- Log in Incident_Register (doc 13) same day
- Reference Order_Log entry
- Escalate to supplier account manager
- Document response
