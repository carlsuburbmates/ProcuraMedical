## Instructions

1. **Incident ID:** Assign sequential number (INC-001, INC-002, etc.)
2. **Date Logged:** Date of entry (YYYY-MM-DD)
3. **Category:** Product/Delivery/Safety/Privacy/Supplier
4. **Severity:** L (Low) / M (Moderate) / H (High)
5. **Order ID:** Link to **Order_Log.md** entry (ORD-XXXX) if applicable
6. **Summary:** 1–2 line description
7. **Immediate Action:** What was done immediately
8. **Outcome:** Resolution or status
9. **Supplier Involved:** Supplier name if applicable
10. **Evidence Location:** Folder path to supporting docs (photos, comms, etc.)
11. **Follow-up Due:** Target date for review/escalation

## Monthly Review

Review all incidents from the prior month to identify:
