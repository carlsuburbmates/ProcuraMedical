## WHAT HAPPENS ON DAY 1

**First order arrives:**

1. Customer fills checkout form (or emails order)
2. You create **Proforma Invoice** (doc 22):
   - Fill in customer, product, price, NDIS code
   - Send to plan manager (if plan-managed)
3. Payment received:
   - Create **Final Invoice** (doc 23)
4. Log order in **Order_Log** (doc 15) immediately:
   - Status: "Pending dispatch"
   - Supplier, SKU, price, margin
5. Contact supplier (email/system):
   - Request blind shipment to customer address
   - Request tracking once dispatched
6. Update Order_Log:
   - Add dispatch date + tracking number
   - Mark delivery when confirmed
7. **If anything goes wrong:**
   - Log in **Incident_Register** (doc 13) same day
   - Escalate to supplier
   - Document all comms
   - Update Supplier_Tracker (doc 16)
